WHAT I'M UP TO

When I'm not making coffee or studying computational theory, I'm working on my homegrown server, a rad pine64 in my living room, and experimenting in web design. This whole site serves as an entry in my portfolio, showcasing trends in my current design work. Lastest additions include some interactive features thanks to the amazing Dark Sky api. I've got plans to integrate some more of that data into a little p5.js module, but you'll have to keep checking back here if you want to see it. If you're interested, you can follow my progress on Github.
