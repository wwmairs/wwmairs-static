WHAT I'M UP TO

When I'm not making coffee or studying cyber security, I'm working on my homegrown server, a rad pine64 in my living room, and experimenting in web design. This whole site serves as an entry in my portfolio, showcasing trends in my current design work. Lastest additions include some interactive features thanks to the amazing Dark Sky api. With a lot of help from the super cool p5.js library, put together a rising and setting sun animation, using data from Dark Sky. If you're interested, you can follow my progress on Github.

This summer some friends and I got very enamored with the card game Hearts. We loved it so much that we played 233 games and logged our scores the whole summer. Want to see the results?

Last spring I tossed together a (bug-prone) lil game about Kanye West with my good friends Avery and Adam. Check it out and let me know if you find some funny bugs or get a highscore.

If you want to see the first site I ever smacked together, you can strut through this portfolio that I made for a project in the web programming class that got this all started.
